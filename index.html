Mobility Ontology

Purpose: Represents AI-generated trip predictions with confidence scores and user associations

Namespace: http://modultech.eu/aicentive/mobility#

Prefix: mob:

Category URI Type Description
CLASSES
Core mob:TripPrediction owl:Class AI-generated trip prediction
Core mob:User owl:Class System user
Activity mob:ActivityType owl:Class Activity type enumeration
Activity mob:BikeActivity Instance Bicycle activity instance
Activity mob:WalkActivity Instance Walking activity instance
Activity mob:CarDriverActivity Instance Car driving activity instance
Activity mob:CarRiderActivity Instance Car passenger activity instance
Activity mob:PublicTransitActivity Instance Public transit activity instance
OBJECT PROPERTIES
Association mob:hasUser owl:ObjectProperty Links prediction to user
Association mob:hasExplanation owl:ObjectProperty Links prediction to explanation
DATA PROPERTIES
Location mob:hasOrigin xsd:string Origin postal code
Location mob:hasDestination xsd:string Destination postal code
Activity mob:hasActivity xsd:string Activity type
Prediction mob:hasPredictedDistance xsd:float Predicted distance (meters)
Prediction mob:hasPredictedDuration xsd:float Predicted duration (minutes)
Confidence mob:hasConfidence xsd:float Confidence score (0-1)
Confidence mob:hasNewRouteConfidence xsd:float New route confidence (0-1)
